# Web 服务

---

## 概述

--

### Web 简介

本课所指的 Web 指代 World Wide Web, WWW 也就是所谓的万维网或者互联网，通俗的说就是“网络出版”中的“网络”。Web 实际上是由 URL 链接起来的 HTML 组成的一个“网”，用户通过计算机使用 HTTP 协议来获取网上的内容也即 HTML。

--

### 应用实例

微信公众号、博客、视频网站、网页游戏以及各种教程网站乃至 Gitbook 等在线书籍都是通过 Web 来为用户提供服务的。

## HTML - <b style='color:red'>H</b>yper<b style='color:red'>T</b>ext <b style='color:red'>M</b>arkup <b style='color:red'>L</b>anguage

--

### 简介

HTML，超文本标记语言，是一种用于创建网页的标准标记语言。它定义了页面的基本结构和内容，可以被浏览器渲染成俗称的“网页”。

HTML 的标准由 W3C（万维网联盟）维护，目前我们所见的大多数网站都是 HTML 4.01 版，而移动互联网时代口头常说的 H5 实际上是 HTML 标准的 5.0 版本。

--

### HTML 元素

相关定义如下：

- HTML 元素以**开始标签**起始，以**结束标签**中止。
- 元素的内容是**开始标签**与**结束标签**之间的内容。
- 有一种元素为**空元素**，在**开始标签**中进行关闭。
- 大多数 HTML 元素可拥有**属性**，如 style。

<!-- .element: style="font-size:70%;" -->

--

### HTML 标签

HTML 文件中，为了区分内容和描述 HTML 文档结构的**标记**，定义了很多**标签**，**标签**由尖括号包裹，分为**开始标签**、**结束标签**以及空标签三种。

```html
<h1></h1>
<p></p>
<br />
```

--

### HTML 设计与构建

类似纸质书排版的过程。

1. 先在草稿纸上画出自己想要制作的页面的大概布局结构。
2. 判断一下各个位置的内容属于 HTML 结构中的什么元素。
3. 查找恰当的标签来填写清楚，构建好自己想要的页面。

<!-- .element: style="font-size:70%;" -->

--

### 图片与超链接

在 HTML 中可以随时添加 `<img/>` 图片元素来加载一幅图片，也可以添加 `<a></a>` 超链接元素来跳转到新的页面。

利用这一特性，可以在数字出版物中加入图片并灵活地组织出版物的内容。

--

### 更多的 HTML 元素

还有更多种类的 HTML 元素和响应的标签需要扩展学习。

更多元素标签，可参考:

https://www.w3school.com.cn/html/html_jianjie.asp 。

### HTML 样式

可以通过 HTML 元素的 style 属性改变元素的表现形式，如颜色、字体、位置等等，或者使用专门的 CSS 样式表来美化页面。

## 常见 HTML 元素

--

### 内容结构

| 元素        | 描述                                                                                                                                                       |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `<h1>-<h6>` | HTML `<h1>–<h6>` 标题(Heading)元素呈现了六个不同的级别的标题，`<h1>` 级别最高，而 `<h6>` 级别最低。                                                        |
| `<p>`       | HTML `<p>`元素或叫做 HTML 段落元素表示文本的一个段落。该元素通常表现为一整块与相邻文本分离的文本，或以垂直的空白隔离或以首行缩进。另外，`<p>` 是块级元素。 |

<!-- .element: style="font-size:70%;" -->

--

### 列表

| 元素   | 描述                                                                                                                                                                                                                                                             |
| ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `<ul>` | HTML `<ul>` 元素或叫做 HTML 无序列表元素表示一个内可含多个元素的无序列表或项目符号列表。                                                                                                                                                                         |
| `<ol>` | HTML `<ol>` 元素表示多个有序列表项，通常渲染为有带编号的列表。                                                                                                                                                                                                   |
| `<li>` | HTML `<li>` 元素 （或称 HTML 列表条目元素） 用于表示列表里的条目。它必须包含在一个父元素里：一个有序列表(`<ol>`)，一个无序列表(`<ul>`)。在菜单或者无序列表里，列表条目通常用点排列显示；在有序列表里，列表条目通常在左边显示按升序排列的计数，例如数字或者字母。 |

<!-- .element: style="font-size:70%;" -->

--

### 文本修饰 (1/2)

| 元素  | 描述                                                                                                                                       |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `<i>` | HTML 元素 `<i>` 用于表现因某些原因需要区分普通文本的一系列文本。例如技术术语、外文短语或是小说中人物的思想活动等，它的内容通常以斜体显示。 |
| `<b>` | HTML `<b>` 是提醒注意元素，用于吸引读者的注意到该元素的内容上。大多数浏览器仍然将文字显示为粗体。                                          |
| `<s>` | HTML `<s>` 元素使用删除线来渲染文本。使用 `<s>` 元素来表示不再相关，或者不再准确的事情。                                                   |
| `<u>` | HTML `<u>` 元素使文本在其内容的基线下的一行呈现下划线。                                                                                    |

<!-- .element: style="font-size:70%;" -->

--

### 文本修饰 (2/2)

| 元素       | 描述                                                                                                                             |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `<em>`     | HTML `<em>` 着重元素标记出需要用户着重阅读的内容， `<em>` 元素是可以嵌套的，嵌套层次越深，则其包含的内容被认定为越需要着重阅读。 |
| `<strong>` | HTML `<strong>` 元素表示文本十分重要，一般用粗体显示。                                                                           |

### 嵌入元素

| 元素     | 描述                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| `<a>`    | HTML `<a>` 元素（或称锚元素）可以创建通向其他网页、文件、同一页面内的位置、电子邮件地址或任何其他 URL 的超链接。 |
| `<br>`   | HTML `<br>` 元素在文本中生成一个换行（回车）符号。此元素在写诗和地址时很有用，这些地方的换行都非常重要。         |
| `<code>` | HTML `<code>` 元素呈现一段计算机代码. 默认情况下, 它以浏览器的默认等宽字体显示.                                  |

<!-- .element: style="font-size:70%;" -->

### 图片和多媒体

| 元素      | 描述                                                                                                                                                                                                                                                                 |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `<audio>` | HTML `<audio>` 元素用于在文档中表示音频内容。 `<audio>` 元素可以包含多个音频资源， 这些音频资源可以使用 src 属性或者<source> 元素来进行描述； 浏览器将会选择最合适的一个来使用。对于不支持<audio>元素的浏览器，<audio>元素也可以作为浏览器不识别的内容加入到文档中。 |
| `<img>`   | HTML `<img>` 元素代表文档中的一个图像。                                                                                                                                                                                                                              |
| `<video>` | HTML `<video>` 元素 用于在 HTML 或者 XHTML 文档中嵌入媒体播放器，用于支持文档内的视频播放。                                                                                                                                                                          |

### 表格

| 元素      | 描述                                                                          |
| --------- | ----------------------------------------------------------------------------- |
| `<table>` | HTML 的 `<table>` 元素表示表格数据 — 即通过二维数据表表示的信息。             |
| `<thead>` | HTML 的`<thead>`元素定义了一组定义表格的列头的行。                            |
| `<tbody>` | 这个 HTML 标签类似 HTML 标签中的 `<body>` 表示表格内容的部分。                |
| `<tfoot>` | HTML 元素`<tfoot>` 定义了一组表格中各列的汇总行。                             |
| `<th>`    | HTML 的 `<th>` 元素代表一个表格的表头部分属性                                 |
| `<tr>`    | HTML `<tr>` 元素定义表格中的行。 元素内包含一系列的 `<td>` 或者 `<th>` 元素。 |
| `<td>`    | HTML `<td>` 元素定义了一个单元格。                                            |
