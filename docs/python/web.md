## 项目 1 - 博客

使用 Flask 框架开发一个个人博客，满足以下要求：

虚拟环境要求：

- 要求使用 pipenv 来管理该项目的环境；
- 批改时我会尝试从纯净环境自动安装依赖。

博客内容要求：

- 一级页面包括三个：首页，文章，关于；
- 二级页面只需要一个，就是点击每篇文章能够跳转到文章具体内容；
- 所有页面布局继承同一个基础模板。

运行要求：

- 批改时我会直接使用 python xxx.py 方式尝试运行项目；
- 请确保能够正常访问你的博客。

_注 1: JavaScript 和 CSS 无特定要求。_

_注 2: 可以用文本文件或者列表、字典直接内嵌文章等数据，不强制使用数据库，用了也没问题。_
