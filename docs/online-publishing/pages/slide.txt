# Pages 网页托管

---

## Pages

--

### 简介

Pages 是 Github 推出的一个网页托管平台，用户可以把某个仓库中存在的 md 转化成 HTML 文件托管在 Github 的服务器上成为一个网站。该服务直接从用户创建的仓库中读取内容并显示为一个网站供他人访问。
