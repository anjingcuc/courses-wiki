# Github 与 Gitee

---

## Github 与 Gitee

--

### 简介

Git 作为一个分布式的版本控制系统，也可以将仓库放在一个所有参与协作的用户都能访问到的服务器上，这种形式的服务我们称之为托管。这个作为中转站的服务器在计算机术语中称为 `Hub` ，所以有一家公司推出了一个网站 `Github` 专门提供仓库托管服务，并开发了许多种额外的功能和服务，本课后半部分用到的 Pages 以及 Gitbook 都是在其中。国内也有类似的服务，就是码云 <https://gitee.com/> 。

--

### 用法

我们可以参考课件下方的教程来进行配置和仓库创建。

---

## Pages

--

### 简介

Pages 是 Github 推出的一个网页托管平台，用户可以把某个仓库中存在的 md 转化成 HTML 文件托管在 Github 的服务器上成为一个网站。该服务直接从用户创建的仓库中读取内容并显示为一个网站供他人访问。

--

### 功能

Pages 使得普通用户只用注册一个 Github 用户即可拥有个性化域名（子域名）并且将自有仓库中的静态网页进行托管成为一个网站，使得个人站点的成本急剧降低。
